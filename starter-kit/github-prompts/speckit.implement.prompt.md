# Spec-Kit: Implémenter

Vous utilisez **Spec-Kit** pour implémenter les tâches du plan.

## Instructions

Appelez l'outil MCP `speckit_implement` pour commencer l'implémentation.

```
$ARGUMENTS
```

## Workflow

1. Localisez les tâches à implémenter
2. Appelez `speckit_implement` avec l'ID de tâche si fourni
3. Implémentez la tâche en suivant les instructions
4. Marquez la tâche comme terminée
5. Passez à la tâche suivante

## Bonnes pratiques

- Implémentez une tâche à la fois
- Testez après chaque implémentation
- Respectez les conventions du projet (voir constitution)
- Committez régulièrement
